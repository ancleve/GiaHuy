Sau khi clone về máy, nếu chưa mở được file "data.rar" thì nhập lệnh "git lfs pull" để tải xuống hoàn tất.
Giải nén file "data.rar" vào trong cùng thư mục trước khi thực hiện các bước build images.
Tất cả các câu lệnh em đều thực hiện trực tiếp trên terminal và có ảnh trong báo cáo các bước thực hiện câu lệnh 
(Do file word bị lỗi hình ảnh nên em xin phép gửi cùng file pdf kèm theo ạ)
